## ⚙️ Comment utiliser ces workflows

1. Télécharge le fichier `workflow.json` du jour qui t'intéresse
2. Dans n8n : nouveau workflow → menu (...) → **Import from File**
3. Configure tes propres credentials (API keys)
4. Active le workflow

Chaque dossier contient son propre guide d'installation détaillé.

## ⚠️ Note

Certains workflows sont adaptés à partir de templates de la communauté n8n, modifiés et configurés pour des cas d'usage réels. Les credentials et clés API ne sont jamais inclus — tu dois ajouter les tiens.
